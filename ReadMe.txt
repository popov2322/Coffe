Для запуска данной утилиты необходимо в командную строку ввести следующие команды:
$cd ~/path/to/this/directory(перемещение в директорию с утилитой необходимо для того, чтобы база данных корректно работала)
$python main.py (*)

				РАБОТА В ИНТЕРАКТИВНОМ РЕЖИМЕ

Для того, чтобы активировать интерактивный режим утилиты, необходимо запустить её без аргументов(т.е. выполнить команды, приведённые выше):
1. Автоматически выведется меню, предлагающее авторизироваться или зарегистрировать нового пользователя. Для выбора необходимо ввести соответствующий команде номер.

	1.) РЕГИСТРАЦИЯ: При регистрации нового пользователя необходимо указать следующие данные:
		а.) Имя пользователя(username) используя символы основной ascii таблицы(на 		английском языке). Например, Ava Taylor, User, Artem 42 Coolguy. Регистрация пользователя 		произведётся только, если данное имя свободно.
		б.) Текущую должность(position): Salesman или Manager. Выбор предоставляется 		только из двух последних =(
	
	2.) АВТОРИЗАЦИЯ: При авторизации необходимо указать только имя пользователя(username), 		зарегистрированного ранее.

2. После авторизации должно появиться командное меню, в котором менеджер может посмотреть статистику продаж, а продавец продавать, товар, узнать цену товара или загрузить чек своей последней продажи.

					Операции:	

	(1) Продажа напитков. Для продажи напитков необходимо ввести сам напиток и добавку к нему(если добавка не нужна, ввести 'none'). После автоматически обновится последний чек продавца и статистика оного
	(2) Узнать цену напитка. Для того, чтобы узнать цену напитка необходимо ввести название напитка.
	(3) Загрузить последний чек.
	(4) Посмотреть статистику.
	(5) Выйти из программы
После выполнение команды утилита спросит желаете ли вы продолжить работу. Для продолжения работы в интерактивном режим необходимо ввести 'y', для завершения работы любую другую клавишу.

				РАБОТА С КЛЮЧАМИ В КОМАНДНОЙ СТРОКЕ

Для выполнения действия в командной строке необходимо запустить команду (*) вместе с ключами:
-h, --help   	     	- команда, отображающая окно помощи, в котором приведены ключи с описаниями их действий
-cmd			- ключ необходимый для работы в режиме коммандной строки
-reg, --registration 	- использовать данный ключ для регистрации пользователя
-u, --username  	- ключ, после которого необходимо ввести имя пользователя
-p, --position  	- ключ, после которого необходимо указать текущую должность(необходимо для регистрации)
-action			- после этого ключа необходимо указать номер выполняем операции, номера операций представлены выше
-b, --beverage		- ключ, после которого необходимо указать напиток, существующий в меню
-a, --additionals	- ключ, после которого необходимо указать добавку, существующую в меню или none если добавка не требуется

Примеры работы с командами: 
Регистрируем нового менеджера:
$python main.py -cmd -reg -u Artem 42 Coolguy -p Manager
Регистрация нового продавца:
$python main.py -cmd -reg -u Vikentsiy Linuxmaster -p Salesman
Менеджер Артём смотрит статистику:
$python main.py -cmd -u Artem 42 Coolguy -action 4
Продавец Викентий продаёт латте с ничего:
$python main.py -cmd -u Vikentsiy Linuxmaster -action 1 -b latte -a none
